## CLI Usage
After installing package `twitch-panels` globally, you can use its command line interface.
```bash
twitch-panels
```
For usage instructions:
```bash
twitch-panels --help
```








## Development



Setting up:
```bash
git clone git@github.com:jaid/twitch-panels.git
cd twitch-panels
npm install
```
Testing:
```bash
npm run test:dev
```
Testing in production environment:
```bash
npm run test
```
